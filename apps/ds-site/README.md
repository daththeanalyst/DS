# @ds/site

The DS company site. Next.js 15 (App Router) + Tailwind 4 + React 19.

## Run it
```bash
pnpm install    # from repo root
pnpm --filter @ds/site dev
```

Then open http://localhost:3000.

## Where the content spec lives
Sitemap + section-by-section copy guidance lives in [`docs/brand/POSITIONING.md` §9 and §10](../../docs/brand/POSITIONING.md#9-website-structure-locked-from-ds2).

This is Phase 3 work. Current scaffold is intentionally minimal — just the hero framing, nav, and footer — so we can deploy early and swap sections in as content is ready.
