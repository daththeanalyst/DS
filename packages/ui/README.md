# @ds/ui

DS's shared React component library. shadcn/ui + Tailwind conventions.

## Rule
Components get added here **when a client project needs them** — not speculatively. Port from [`packages/frontendmaxxing-reference`](../frontendmaxxing-reference/) (vanilla JS/CSS source of truth) into React equivalents, applying DS design principles (see [`.claude/skills/frontend-design/SKILL.md`](../../.claude/skills/frontend-design/SKILL.md)).

## First 10 ports (when the first client arrives)
Hero · Nav · Pricing · Testimonials · FAQ · Footer · Card · Modal · Form · Marquee.
